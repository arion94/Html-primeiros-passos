Passo a Passo

Primeiramente é necessário entender a comunicação entre cliente e servidor.

O cliente é o consumidor da informação web.

Servidor é um computador central que processa e armazena o que esta sendo pesquisado em outros computadores, tendo a comunicação direta com a internet

    Tipos de servidores:
  • Proxy
  • Firewall
  • Database
  • DNS
  • Hospedagem de sites
    REsumindo a explicação:
  1. O servidor proxy:Barreira colocada no acesso da internet para o computador que autoriza ou desautoriza o acesso a daterminado tipo de site
  2. servidor Firewall : Barreira de segurança para impedir atques externos
  3. Database server : Servidor que é acessado pelo usuário visando buscar banco de dados
  4. DNS : Servidor que da nome a um site de internet, pois senão as pesquisas se dariam apenas pelo númnero IP
  5. Hospedagem de sites : servidor que hospeda um site
